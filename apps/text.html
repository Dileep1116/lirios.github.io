About

Liri Text is a cross-platform, Material Design text editor, currently in  alpha  stage.

We aim to create a minimalistic and simple text editor providing all the features you would expect from a modern editor. Following Google's Material Design allows us to make interface clean and beautiful.

Features

Straightforward

An editor that doesn't interfere with your work

Liri Text has a simple and straightforward UI so you can focus on your task.

Solid

An editor that lets you do more

In Liri Text you'll find all the basic editing features. Whether you want to modify a config file or to write a script, Liri Text will not fail you.

Syntax Highlighting

An editor that makes reading code easier

Liri Text inludes a syntax coloring system that supports a wide range of languages. It does so by supporting GtkSourceView language definitions.

Contribute

Liri Text is community-driven open-source project. We rely on contributions to keep the project sustainable. Don't think you can only contribute if you know how to code! There are plenty of ways to help out:

Design

Everything starts with an idea. If you're a designer, your contribution in form of mockups, sketches or other graphics are from great value for the whole project. Checkout the GitHub issue list and get in touch by joining our Community on Google+.

Code

Writing code is without any doubt a fantastic way to contribute to the development of the project. Liri Text is written using Qt framework in C++ and QML languages. We use QtQuick Controls 2 and Fluid to create our Material Design based user interface.

git clone https://github.com/lirios/text

Your pull requests are always welcome!
